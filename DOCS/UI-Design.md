# UI Design

ASMRPlayer 当前界面以 `DOCS/design/exports/asmr-player-app-standalone.html` 展示的应用效果为视觉基准，并以 `DOCS/design/tokens/`、`DOCS/design/ui-kits/app/*.jsx` 及 `DOCS/design/components/` 中的设计系统作为实现参照。

## 设计资产索引
- `DOCS/design/exports/asmr-player-app-standalone.html`：应用主界面原型（单页）
- `DOCS/design/styles.css`：主样式变量与页面层级
- `DOCS/design/tokens/`：字体、色彩、间距、阴影、圆角和动效令牌
- `DOCS/design/guidelines/`：配色、字体、动效、半径与层级规范文档
- `DOCS/design/components/`：按钮、开关、搜索、进度条、卡片、媒体展示等组件快照与文档
- `DOCS/design/ui-kits/`：应用关键 Screen 原型（`readme.md`、`index.html`、原型页面 `*.jsx`）
- `DOCS/design/assets/`：logo 等可复用视觉资源

## 全局视觉

- 主题基调：安静、柔和、低干扰；深色界面使用黑色页面底色和浅黑 `#17181A` 设置行层级，浅色界面使用浅灰页面底色和白色设置行层级。
- 主强调色：中性灰用于播放、选中、焦点、进度和关键动作，避免暖黄色在黑白主题中跳出。
- 次强调色：`mist` 薰衣草用于次级提示、AI/信息辅助状态。
- DLsite 专属强调：DLsite 页面保留 `#4F9BE0` 作为账号、标识等品牌色；作品卡与内容行的操作图标、展开箭头和下载状态使用全局中性灰强调。
- 字体映射：Android 端使用系统 Sans/Serif/Mono 对应设计包的 Hanken Grotesk、Instrument Serif、JetBrains Mono；标题和播放器主字幕优先使用 Serif 风格，编号、时间和进度信息优先使用 Mono 风格。
- 形态规则：卡片使用 20-36dp 大圆角，按钮和 chip 接近 pill；页面卡片在深色下使用浅黑 surface，在浅色下使用微灰白 surface，配合细边框和低调阴影。
- 弹出菜单使用主题 sheet 背景、细边框和 20dp 圆角，不使用系统默认方角或默认色。
- 动效规则：展开、切换、播放状态高亮使用慢速柔和动画；播放中状态使用 `WaveBars`，播放时竖条错峰循环，暂停时保持低高度。
- 图标规则：主要导航、播放器和状态图标使用 `DOCS/design` 中的 Lucide outline 风格；Android 无直接等价图标时使用 Compose Canvas 复刻。

## 应用壳

- 顶部标题只显示加大号 Serif 标题，右侧菜单只保留裸三点图标，不显示外框或填充色；各 tab 顶部不显示 eyebrow 小字。
- 底部区域由 glass mini player 和四个 tab 组成。
- Tab 顺序固定为：`资料库`、`睡眠定时`、`DLsite`、`设置`。
- 底部 Tab 选中态使用粉色图标和文字，未选中态保持弱化灰色。
- Mini player 显示 44dp 封面、当前音频、上下文和播放/暂停；使用无阴影的 glass surface 与细边框；仅播放中在播放按钮旁显示 `WaveBars` 波浪动效；点按后播放器以不透明实体面板从 mini player 所在底部区域向上展开，关闭时向底部收起，不做半透明叠加。
- 全屏播放器作为覆盖层打开，不改变底层 tab 导航结构；覆盖层会消费整屏触摸，空白区域点击不透传到底层列表或 tab 页面。

## 资料库

- 页面顶部为搜索框，下面显示播放列表 accordion。
- 播放列表卡片显示封面、标题、曲目数量和播放状态；点按卡片展开或收起。
- 展开后显示分段/曲目列表，曲目保留字幕、AI 字幕进度、重命名、删除、移动等现有操作。
- 当前播放作品卡片使用粉色标题/状态文字表示选中；律动条复用 mini player 的 `WaveBars`，且仅在正在播放时显示。展开曲目行的当前项不使用框或填充色，只把序号和曲目标题文字改为粉色。
- 没有可展示内容时显示 DLsite 连接空态和连接 CTA。
- 搜索只过滤本地已有播放列表和曲目，不改变播放列表数据。

## 播放器

- 全屏播放器优先从当前播放列表封面提取主色生成背景底色和两层氛围光；没有封面或封面解码失败时，深色下使用纯黑背景和灰色/mist 氛围光，浅色下使用白色背景与轻量层级。
- 顶部栏为裸下拉关闭图标、当前播放列表名和右上裸三点更多操作入口；不显示固定的 `正在播放` 文案，没有播放内容时中间标题留空；关闭和三点入口都不显示外框或填充色。
- 中部为字幕展示区：区域背景与全屏播放器背景保持一致，不使用独立色块、边框、矩形遮罩或额外律动竖线；当前行居中高亮，前后行弱化显示；没有字幕内容时不显示 `未载入字幕` 或 `无字幕` 占位。
- 底部显示小封面、当前标题、进度条、剩余时间、随机、上一首、播放/暂停、下一首、循环；上一首、播放/暂停、下一首使用无外框裸图标按钮，并保持相同尺寸和颜色；当前标题强制单行展示，过长时用省略号截断，不显示下方上下文小字。
- 底部动作固定为睡眠、混音、队列；按钮与上一首/播放/下一首保持同样的无外框裸图标样式、尺寸和颜色，不显示文字；已有真实业务的入口接真实回调，暂未实现的入口保留按钮并给出占位提示。
- 右上更多菜单以底部 sheet 展示 AI 字幕、睡眠定时、混音层叠、下载到本地/已下载、从队列移除；AI 字幕入口接当前字幕生成服务或进度 sheet。
- 播放队列以底部 sheet 显示，包含 grabber、当前播放和接下来列表。

## 睡眠定时

- 中央使用 210dp 圆形 dial 展示剩余时间或睡眠状态；默认没有活动定时且未点预设时，圆环无填充、中心不显示倒计时；普通倒计时只显示在圆环中心，格式为 `mm:ss`，不显示小号“分钟”文字，圆环下方不保留任何状态文案区域；环形进度在开始定时后填满，并随剩余时间按逆时针方向流逝，中心倒计时数字、圆环进度和播完当前作品图标在暗色主题使用白色、浅色主题使用深色前景。
- 预设分钟 chip 横向展示：15、30、45、60、90 分钟；默认不选中任何预设；选中态使用粉色背景和白色文字，点击任意预设会立即开始对应分钟倒计时。
- 设置卡只显示结束前淡出、播完当前作品后停止两个开关；开关行只显示标题和右侧开关，不显示小字说明。
- 预设分钟和播完当前作品入口会直接启动睡眠定时；底部不显示“开始睡眠定时”主按钮。
- 底部只保留一个复用文字入口：没有活动定时时显示“自定义时间”，定时启动后同一位置变为“取消定时”。

## DLsite

- 未登录态显示 DLsite 品牌同步卡、账号/密码输入框视觉、登录并同步按钮、安全说明和注册链接；登录仍通过现有 DLsite 登录 Activity 完成。
- 已登录态显示账户卡、同步状态、统计信息和当前 DLsite 内容清单。
- 同步、内容解析、内容下载、暂停、继续、删除和下载管理均保留当前真实回调。
- 当前 DLsite 内容清单展示 `state.works` 中的全部作品；每个作品卡显示封面、标题、RJ 编号和解析/展开入口，不展示作品级“已载入/部分载入/载入中”等整体下载状态。
- 未解析内容的作品卡右侧显示小号同步图标按钮，不显示文字；点击后解析并缓存 DLsite 可下载目录，不弹出二级内容选择页。
- 已解析内容的作品卡采用类似资料库播放列表的 accordion 交互，右侧显示展开/收起箭头，不再显示“已解析”文字；点按作品卡头部展开或合上内容清单，内容下载中或下载完成后仍保持同一套 accordion 目录样式。
- 展开后的作品卡在卡片内展示 `contentsByWork` 中该作品的全部下载内容；内容级下载任务不会把作品卡头部切换成整卡下载态或整体已下载态，正在下载的内容项在自身行下方显示进度线，方便继续下载其它内容；内容行左侧原音乐图标位置显示下载状态勾选图标，未下载为灰色，已下载使用全局强调色，右侧只保留单行下载/删除/重试/暂停/继续或进行中状态。
- DLsite 页面只在品牌标识和账号连接感知上使用蓝色，其余操作图标、按钮状态和展开控制遵循全局中性灰强调。

## 设置

- 设置页使用分组卡片结构，顶部大标题左对齐；主设置页不显示播放、下载、外观、字幕、权限、AI、关于等分组 label 小字。
- 现有卡片顺序按播放、下载、外观、字幕、权限、AI、关于展示；播放卡片只保留与本地播放体验相关的开关，不展示独立播放音质档位；外观卡片的主题标题与分段控件保持同一行；关于卡片只保留版本与更新检查，更新检查空闲态不显示“未检查”占位文案。
- 设置 tab 不使用外层共享标题栏，主设置页和 AI 设置页各自拥有自己的顶部区域；AI 设置为独立二级页，顶部栏在同一行显示返回箭头和大号「设置」，不显示返回旁“设置”小字或「AI 字幕与翻译」大标题，并保留当前翻译引擎、转写后端、Whisper 模型、远程 Whisper、API Key 和测试连接入口。
- 设置页进入 AI 二级页时，二级页从右侧推入并把主设置页向左推出；返回时方向相反，主设置页恢复到离开前的滚动位置，不回到顶部。
- 设置项左侧图标使用粉色线性图标，不加图标底框；开关行保持左侧粉色图标、中间标题、右侧 switch 的结构。
- 带左侧图标的设置行统一使用相同的图标列和文本起点，开关行、操作行、值行、选择行标题必须左对齐。
- 设置项标题字体统一使用设置操作行样式：15sp、SemiBold；右侧值和状态文字统一使用 13sp。
- 关于分组作为例外不显示左侧图标；检查更新行不显示右侧箭头，只保留可点击行本身和状态文字。
- Switch 使用系统设置式视觉：开启态为粉色轨道和白色圆点，关闭态为灰色轨道和白色圆点；设置开关行使用纯文本标题、右侧开关和细分隔线。
- 主题切换分段控件的选中项使用粉色椭圆描边，不使用内部填充色；点击选项时不显示灰色按压框，选中描边在选项之间平滑移动，选中文字同步使用粉色。
- Segmented control、进度卡和更新卡统一使用随主题变化的浅黑/微灰白 surface、pill 选中态和中性灰强调。
