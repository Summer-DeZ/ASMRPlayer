# UI Design

ASMRPlayer 当前界面以 `DOCS/ASMRDesign/exports/ASMRPlayer-App-standalone.html` 展示的应用效果为视觉基准，并以 `DOCS/ASMRDesign/tokens/`、`DOCS/ASMRDesign/ui_kits/app/*.jsx` 中的设计系统作为实现参照。

## 全局视觉

- 主题基调：深夜、安静、柔和；默认深色界面使用近黑 `ink` 背景。
- 主强调色：`ember` 暖橙用于播放、选中、焦点、进度和关键动作。
- 次强调色：`mist` 薰衣草用于次级提示、AI/信息辅助状态。
- DLsite 专属强调：DLsite 页面使用 `#4F9BE0` 蓝色作为局部品牌色。
- 字体映射：Android 端使用系统 Sans/Serif/Mono 对应设计包的 Hanken Grotesk、Instrument Serif、JetBrains Mono；标题和播放器主字幕优先使用 Serif 风格，编号、时间和进度信息优先使用 Mono 风格。
- 形态规则：卡片使用 20-36dp 大圆角，按钮和 chip 接近 pill；页面卡片使用暗色 surface、细边框和低调阴影。
- 动效规则：展开、切换、播放状态高亮使用慢速柔和动画；播放中状态使用 `WaveBars`，播放时竖条错峰循环，暂停时保持低高度。
- 图标规则：主要导航、播放器和状态图标使用 ASMRDesign 中的 Lucide outline 风格；Android 无直接等价图标时使用 Compose Canvas 复刻。

## 应用壳

- 顶部标题使用小号大字距 eyebrow 加大号 Serif 标题，右侧菜单为圆形图标按钮。
- 底部区域由 glass mini player 和四个 tab 组成。
- Tab 顺序固定为：`资料库`、`睡眠定时`、`DLsite`、`设置`。
- Mini player 显示 44dp 封面、当前音频、上下文和播放/暂停；仅播放中在播放按钮旁显示 `WaveBars` 波浪动效；点按打开全屏播放器。
- 全屏播放器作为覆盖层打开，不改变底层 tab 导航结构。

## 资料库

- 页面顶部为搜索框，下面显示播放列表 accordion。
- 播放列表卡片显示封面、标题、曲目数量和播放状态；点按卡片展开或收起。
- 展开后显示分段/曲目列表，曲目保留字幕、AI 字幕进度、重命名、删除、移动等现有操作。
- 当前播放作品和曲目使用 ember 背景、强调文字和 `WaveBars`/audio-lines 风格动效标记。
- 没有可展示内容时显示 DLsite 连接空态和连接 CTA。
- 搜索只过滤本地已有播放列表和曲目，不改变播放列表数据。

## 播放器

- 全屏播放器使用近黑背景和 ember/mist 氛围光。
- 顶部栏为下拉关闭、`正在播放` 状态和右上 `ellipsis` 更多操作入口。
- 中部为字幕展示区：当前行居中高亮，前后行弱化显示。
- 底部显示小封面、当前标题、上下文、进度条、剩余时间、随机、上一首、播放/暂停、下一首、循环。
- 底部动作固定为睡眠、混音、队列；已有真实业务的入口接真实回调，暂未实现的入口保留按钮并给出占位提示。
- 右上更多菜单以底部 sheet 展示 AI 字幕、睡眠定时、混音层叠、下载到本地/已下载、从队列移除；AI 字幕入口接当前字幕生成服务或进度 sheet。
- 播放队列以底部 sheet 显示，包含 grabber、当前播放和接下来列表。

## 睡眠定时

- 中央使用 210dp 圆形 dial 展示剩余时间或睡眠状态。
- 预设分钟 chip 横向展示：15、30、45、60、90 分钟。
- 设置卡显示结束前淡出、播完当前作品后停止、入睡音量。
- 开始睡眠定时使用底部主按钮；真实定时回调接入预设分钟/播完当前/自定义，淡出和音量先作为本地 UI 状态展示。
- 激活定时后显示取消入口。

## DLsite

- 未登录态显示 DLsite 品牌同步卡、账号/密码输入框视觉、登录并同步按钮、安全说明和注册链接；登录仍通过现有 DLsite 登录 Activity 完成。
- 已登录态显示账户卡、同步状态、统计信息和当前 DLsite 内容清单。
- 同步、下载、暂停、继续、删除、下载管理和下载内容选择均保留当前真实回调。
- 当前 DLsite 内容清单展示 `state.works` 中的全部作品；每个作品卡显示封面、标题、状态、载入状态操作和进度条。
- 每个作品卡默认展示 `contentsByWork` 中该作品的全部下载内容，内容行展示标题、状态、音频数量和本地缓存状态；未解析下载内容时提供“尚未解析下载内容 / 查看”入口，继续触发现有内容选择 sheet。
- DLsite 页面只在局部使用蓝色品牌强调，其余仍遵循全局 ASMRDesign dark tokens。

## 设置

- 设置页使用分组卡片结构，分组 label 使用小号大字距样式。
- 现有分组按设计展示为播放、下载、外观、AI、关于；设计要求的按钮入口完整显示，暂未实现的播放/下载/隐私政策入口使用占位提示。
- AI 设置为二级页，保留当前翻译引擎、转写后端、Whisper 模型、远程 Whisper、API Key 和测试连接入口。
- Segmented control、switch、进度卡和更新卡统一使用暗色 surface、pill 选中态和 ember 强调。
